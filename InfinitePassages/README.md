# Infinite Passages

In Story mode, prevents Passage Tokens from being consumed and restores any that are already consumed, and optionally skips the Passage Token animation to jump straight to the Fast Travel menu (enabled by default).

In Expedition mode, prevents earned Passages from decreasing.

Also, reduces the Fast Travel's Cycles cost from 5 to 1, and optionally prevents max Karma regain on using Passage (fast travel) (enabled by default).
